Summary

I will implement a parallel version of the CKY algorithm used for parsing natural language.

Progress

So far, I have adapted a barebones serial implementation of the CKY algorithm to be able to read in the Berkeley grammar and lexicon. I have also created a simple testing harness that I will evolve as I continue development.
